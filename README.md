
# NourishIQ

## AI-Powered Nutrition Intelligence for Children

Predictive infrastructure that helps governments, NGOs, and schools identify and respond to child hunger before it impacts education, health, and development.

---

# Overview

NourishIQ is a production-ready social impact SaaS dashboard that combines AI-driven analytics with real-world nutrition data to predict child hunger-risk zones across districts and schools.

The platform monitors:
- school attendance patterns
- midday meal participation
- local food price inflation
- regional social indicators

to generate actionable intelligence for intervention planning.

---

# Key Features

## Hunger Heatmap
Interactive geographic visualization of child hunger-risk clusters across monitored districts.

## Predictive Analytics
AI-generated risk scores, trend forecasting, and anomaly detection.

## District Intelligence
Detailed district-level drill-down analytics with school performance insights.

## Intervention Tracking
Monitor nutrition support programs and intervention impact metrics.

## School Onboarding
Streamlined workflow for integrating new schools into the monitoring network.

## AI Report Generation
Automated intelligence reports for NGOs, governments, and stakeholders.

## Fully Responsive
Optimized dashboard experience across desktop, laptop, tablet, and mobile devices.

# Getting Started

## Installation

```bash
# Clone repository
git clone https://github.com/Garv0714/nourishiq.git

# Open project
cd nourishiq

# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build production version
pnpm build
````

# Vision & Impact

## Problem

Millions of children silently attend school while suffering from hunger and malnutrition. Most systems react only after attendance drops and educational outcomes decline.

## Solution

NourishIQ acts as an early-warning nutrition intelligence system by combining attendance trends, meal participation, and regional indicators to identify child hunger-risk zones before the situation becomes critical.
